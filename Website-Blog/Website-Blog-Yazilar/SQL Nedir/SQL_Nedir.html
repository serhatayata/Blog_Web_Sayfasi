SQL Nedir ve Nasıl Kullanılır?

SQL (Structured Query Language) kullanımı veri tabanı olarak değil, veritabanının içindeki düzenleyicilerden bir tanesi olarak kullanılmaktadır. SQL kullanımının amacı, veritabanı içerisinde yer alan verilerin düzene koyulması ve kolay şekilde yönetilmesini sağlamaktadır.

SQL Kullanımının Avantajları ve Özellikleri

  • Kullanıcıların kolaylıkla verilere erişmesini sağlamaktadır.
  • Veriler için açıklama hazırlanmasına ve kolayca kategorize edilmelerine izin vermektedir.
  • Kullanıcılarının, veriler üzerinde değişiklik yapmaları için farklı sistemlere sahiptir ve bu sistemler sayesinde, veriler her zaman güncel tutulur.
  • SQL veritabanı uygulaması, tescilli şekilde hazırlanmıştır ve açık kaynak olarak hizmet vermektedir.

SQL Server Management Studio Kullanımı Nasıl Olmalıdır?

SQL Server programı üzerinden, SQL sunucusuna erişilmek ve bu veritabanı üzerinden gerekli işlemleri gerçekleştirmek mümkündür. Başlat menüsü üzerinden program çalıştırıldıktan sonra “Connect to Server” ekranı açılmaktadır. “Server Name” kısmı üzerinden bağlantı kurulmak istenen sunucu modeli seçilmelidir. Sunucu seçimi yapıldıktan sonra, “Windows Authentication” yönetimi tercih edilmelidir. “Connect” butonuna tıklandıktan sonra, bilgisayar üzerinden SQL sunucusuna erişim sağlanmaktadır.

SQL Server Veritabanı Oluşturmak İçin Yapılması Gerekenler

SQL sunucusunda yer alan nesnelerin tamamına erişim sağlamak için sol panelde yer alan Object Explorer bölümü kullanılmaktadır. Panelde yer alan “Databases” simgesi üzerine iki defa tıklandığı yer, bağlı bulunan sunucu üzerindeki veritabanının tamamının yer aldığı bir liste sıralanacaktır. Erişilmek istenen veritabanı seçimi yapıldıktan sonra, veritabanı üzerine çift tıklanmalıdır. Eğer sıfırdan veritabanı oluşturulacaksa “Databases” simgesinin üzerine çift tıkladıktan sonra “New Database” bölümü üzerinden talimat verilmelidir. Açılacak pencere üzerinden “Database Name” seçimi yapıldıktan sonra veritabanının ismi yazılmalı ve “Ok” butonuna tıklanmalıdır.

SQL Yazılımlarının Türleri ve Özellikleri Nelerdir?

  • MySQL : Oracle isimli firma tarafından üretilmiştir. Bugün için en fazla tercih edilen SQL yazılım modeli arasında yer almaktadır. Açık kaynak kodlarına sahiptir ve tüm lisansları ücretsizdir. MySQL, çifte lisansa sahiptir ve sahip olduğu ikinci lisans modeli GPL lisansıdır.
  • MSSQL : Ücretli olarak tasarlanmış lisans türüdür. Hosting firmaları, talep edilen ücretleri karşılamaktadır. .NET ya da ASP dilleriyle üretilen internet siteleri için idealdir.
  • Microsoft SQL Server : Microsoft tarafından geliştirilen sunucu yazılımı modelidir. Program kapalı kaldığı süreler içerisinde, herhangi bir sorun oluşturma ve tüm yazılım planlamalarını kendi başına devam ettirir.
  • Firebird: İçerisindeyer alan Alert özelliği sayesinde veritabanında bulunan tabloların tüm hatalarını düzeltmektedir. Tablolardaki farklılıkların düzenlenmesi için farklı yazılımların kullanılmasına gerek kalmaz. Türkçe dil desteği bulunur ve hizmetlerinin tamamı ücretsizdir.
  • Access : Microsoft tarafından geliştirilen veritabanı modellerinden bir tanesidir. Çoğunlukla mikro ölçekli uygulamaların geliştirilmesinde kullanılmasında kullanılmaktadır. Mobil uyumludur ve taşınabilir bir sisteme sahiptir. SQL yazılımına sahip olmayan kişilerin, veritabanı içinde yer alan raporlara erişimine izin vermektedir.

    • SQL çeşitleri arasından en uygun model seçildikten sonra, lisans anlaşması yapılmalı ve veritabanı yüklemeleri gerçekleştirilmelidir.
